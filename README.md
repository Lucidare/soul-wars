# RuneLite Soul Wars Tracker Plugin

A RuneLite plugin to track in-game progress during the Soul Wars minigame, keeping a count of:
- Fragments sacrificed
- Avatar damage dealt
- Bones buried
- Areas captured

Set the target goals in the config.
